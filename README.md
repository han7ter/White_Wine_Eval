# White Wine Quality Predictor
Проект по оценке белого вина моделью на основе 11 параметров
# Описание
Этот проект использует машинное обучение для предсказания оценки качества белого вина (по шкале от 0 до 10) на основе набора химических параметров. Модель обучается на датасете, содержащем информацию о различных образцах вина.
Технологии: Python, PyTorch
